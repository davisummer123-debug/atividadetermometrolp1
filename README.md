# atividadetermometrolp1
Estrutura do Repositório
Neste repositório, você encontrará as resoluções da atividade de Termômetro divididas por questões:

Questão 01: Impressão de Números de 1 a N

Questão 02: Cálculo de Fatorial

Questão 03: Soma de Números em um Array

Questão 04: Conversão de Temperatura

Questão 05: Contagem de Caracteres em uma String

Questão 06: Verificação de Número Primo


🚀 Como executar os códigos

Entrar na pasta da questão desejada:
cd Questao_01

Compilar o código:
gcc questão1.c -o exec_questao1

Executar:
./exec_questao1
